Une API(application programming interface) est une interface qui permet a un serveur front-end et un serveur back-end de communiquer. 

Les API peuvent se comporter différemment en fonction de la requête de l'utilisateur.

--------------------------------------
Pour détecter une API, l'on peut scanner l'application web avec un outil comme OWASP [[ZAP]] afin de détecter un endpoint `/api/` , ou encore du JavaScript qui pourrait faire référence a un endpoint caché.
