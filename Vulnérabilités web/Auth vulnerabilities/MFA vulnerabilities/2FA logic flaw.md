Sur certaines applications, la MFA peut être bypass autrement. L'acteur malveillant peut se connecter a son propre compte, trafiquer les requêtes HTTP pour que celles-ci envoient un code de vérification a la victime.

Ensuite, l'attaquant devra bruteforce ce code, dès l'instant où il connait sa taille.

L'attaquant utilise donc son compte pour passer l'authentification, envoie un code de vérification a la victime, et il bruteforce celui-ci.
