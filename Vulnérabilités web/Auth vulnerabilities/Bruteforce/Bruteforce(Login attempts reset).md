Pour certaines app web, une machine est bloquée si elle envoie trop de requêtes qui échouent. Cependant, elle réinitialise le compte de requêtes de cette machine si elle envoie une qui fonctionne.
Autrement dit, un attaquant pourrait donc inclure ses propres identifiants dans une wordlist afin de réinitialiser le compte de façon fréquente.

Note: Le blocage de comptes est aussi un vecteur d'énumération.
